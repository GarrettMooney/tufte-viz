tufte-viz

A demo gallery for the tufte-viz Claude Code skill — data visualizations built on Edward Tufte's principles. Skill by aparente.

  1. NASA GISS temperature anomalies
    Global surface temperature anomalies, rendered as small multiples by latitude band.
  2. Kyoto cherry blossom · 812–2015
    Twelve centuries of peak-bloom dates — a millennial climate record in one line.
  3. Sunspot butterfly diagram
    Solar latitude of sunspots over time — the classic Maunder butterfly.
  4. Sunspots · austere vs. beautiful
    The same sunspot data, austere versus decorated — a study in data-ink.

Install in Claude Code

# add this repo as a plugin marketplace, then install
/plugin marketplace add GarrettMooney/tufte-viz
/plugin install tufte-viz@tufte-viz
